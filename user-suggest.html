가족 비용 추적 앱이 준비되었습니다. 이 가이드는 앱을 최대한 활용하기 위한 다음 단계를 제시합니다.

현재 상태: MVP 완료 (8개 페이지, 모든 핵심 기능 구현)

🚀 1단계: 빠른 시작

설치 & 실행

$ npm install
$ npm run dev
→ localhost:3000

첫 번째 그룹 만들기

  1. ✓ 대시보드 방문 (http://localhost:3000/dashboard)
  2. ✓ "새 그룹 만들기" 클릭
  3. ✓ 그룹명 입력 (예: 가족, 여행)
  4. ✓ 멤버 2명 이상 추가
  5. ✓ "그룹 생성" 클릭

✨ 2단계: 핵심 기능 활용

💰

지출 기록

그룹 상세에서 "지출 추가" → 항목명, 금액, 결제자, 참여자 선택 → 자동 정산 계산

📊

대시보드 & 통계

대시보드에서 전체 지출, 월별 추이, 카테고리별 분석 확인

🔍

검색 & 필터

지출 목록에서 항목명 검색, 카테고리/날짜 필터로 상세 조회

💸

정산 관리

"정산" 탭에서 누가 누구에게 얼마를 줄지 확인 → "정산 완료" 표시

설정 페이지에서 "데이터 내보내기" → CSV 파일 다운로드 → 엑셀에서 열기

❓ 자주 묻는 질문

데이터는 어디에 저장되나요?

모든 데이터는 로컬 SQLite 데이터베이스에 저장됩니다 (클라우드 없음). 데이터는 당신의 컴퓨터에만 있습니다.

그룹에 다른 사람을 추가할 수 있나요?

현재는 같은 컴퓨터에서만 사용 가능합니다. 향후 그룹 공유 링크/QR 코드 기능을 추가할 예정입니다.

정산 계산은 어떻게 작동하나요?

Greedy 알고리즘을 사용해 최소 거래 수로 모든 사람이 정산되도록 계산합니다. 예: A가 ₩10,000 더 지출했고 B가 ₩5,000 더 지출했다면, C가 A에게 ₩10,000, B에게 ₩5,000을 주는 대신 C가 A에게 ₩15,000을 주면 끝!

백업은 어떻게 하나요?

설정 페이지에서 "데이터 내보내기"로 CSV 파일을 다운로드해 정기적으로 백업하세요. 향후 자동 백업 기능을 추가할 예정입니다.
